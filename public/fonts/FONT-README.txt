IMPORTANT: Add Halimum Font

The Halimum script font file (halimum.woff2) needs to be placed in this directory.

If you don't have this font file, you can:
1. Use an alternative script font from Google Fonts
2. Download a similar font and convert it to WOFF2 format
3. Update the font fallback in app/layout.tsx to use a different script font

Recommended alternatives:
- Dancing Script
- Pacifico
- Great Vibes
- Allura

To use a Google Font instead:
1. Import it in app/layout.tsx
2. Remove the localFont declaration
3. Update the font variable
